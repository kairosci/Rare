# RareStyle Stylesheet

This directory contains the Material Design-inspired stylesheet for Rare with cyan colors from the logo.

## File Structure

- **`template.qss`** - Base QSS template with color scheme (generated by `generate_material_style.py`)
- **`stylesheet.py`** - Python script that reads template.qss and generates stylesheet.qss
- **`stylesheet.qss`** - Final compiled stylesheet (AUTO-GENERATED - DO NOT EDIT)
- **`__init__.py`** - Compiled Qt resources (AUTO-GENERATED - DO NOT EDIT)
- **`generate_material_style.py`** - Script to generate Material Design template with cyan colors
- **`stylesheet.qrc`** - Qt resource file listing all assets

## Workflow for Style Changes

### Option 1: Change Colors (Recommended)

If you want to modify colors:

1. Edit `generate_material_style.py` and update the color mappings
2. Run the script to regenerate `template.qss`:
   ```bash
   python3 generate_material_style.py
   ```
3. Regenerate compiled files:
   ```bash
   python3 stylesheet.py
   ```

### Option 2: Direct Template Editing

If you need to make structural CSS changes:

1. Edit `template.qss` directly
2. Regenerate compiled files:
   ```bash
   python3 stylesheet.py
   ```

### Option 3: Use Build Tools

From the repository root, you can use:
```bash
bash tools/qrc2py.sh --force
```

This will regenerate all stylesheet resources across the project.

## Color Scheme

The current color scheme is based on the cyan color from the Rare logo:

- **Primary Cyan**: `rgb(45, 172, 206)` / `#2dacce` - Main borders and accents
- **Highlight Cyan**: `rgb(65, 192, 226)` / `#41c0e2` - Lighter cyan for highlights
- **Selection**: `rgb(35, 85, 103)` / `#23556f` - Cyan-tinted selection background
- **Hover**: `rgb(30, 40, 45)` / `#1e282d` - Slightly cyan-tinted hover background

## Material Design Principles Applied

- **Rounded Corners**: 4px border radius for smoother visual flow
- **Cyan Accents**: Logo color used for interactive elements
- **Consistent Spacing**: Uniform padding and margins
- **Clear Hierarchy**: Color and elevation to distinguish UI layers

## Important Notes

⚠️ **DO NOT manually edit these auto-generated files:**
- `stylesheet.qss` (generated by `stylesheet.py`)
- `__init__.py` (generated by `stylesheet.py`)

✅ **Edit these files to make changes:**
- `generate_material_style.py` (for color scheme changes)
